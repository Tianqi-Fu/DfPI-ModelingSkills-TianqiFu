# DfPI Modeling Skills Portfolio

This repository contains the Modeling Skills portfolio files, including Rhino, Processing, ComfyUI workflow files, and references to the final video output.

## Repository Contents

- `01_rhino/` Rhino modeling files
- `02_processing/` Processing sketches and scripts
- `03_comfyUI/` ComfyUI workflow files and related outputs
- `04_final video/` final video output reference
